Анастасия — Scoping && Hoisting

Scoping:

LOCAL-Локальные переменные доступны только изнутри функции или блока

GLOBAL-Глобальные переменные доступны откуда угодно. Глобальная область видимости — это механизм, который позволяет среде выполнения JavaScript-кода (браузеру, Node.js) предоставлять приложениям хостовые (т.е. принадлежащие среде) объекты как глобальные переменные. Например, window и document являются глобальными переменными (объектами), предоставляемыми браузером

Hosting:

Поднятие или hoisting — это механизм в JavaScript, в котором переменные и объявления функций, передвигаются вверх своей области видимости перед тем, как код будет выполнен.

Что следует помнить:

✓ JavaScript имеет глобальную и локальную область видимости.

✓ Переменные, объявленные и инициализированные вне любой функции, становятся глобальными переменными.

✓ Переменные, объявленные и инициализированные внутри функции, становятся локальными переменными для этой функции.

✓ Переменные, объявленные без ключевого слова const / let внутри любой функции, автоматически становятся глобальными переменными.

✓ К глобальным переменным можно получить доступ и изменить их в любом месте программы.

✓ Доступ к локальным переменным невозможен вне объявления функции.

✓ Глобальная переменная и локальная переменная могут иметь одно и то же имя, не влияя друг на друга.

✓ JavaScript не допускает использование области уровня блока внутри скобок {}.

✓ Запомните и держите в уме одну важную деталь, JavaScript непреклонно сначала объявляет, а уже затем инициализирует наши переменные.
